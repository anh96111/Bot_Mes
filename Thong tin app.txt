📋 MÔ TẢ DỰ ÁN: BOT_MES - Facebook Messenger Dashboard
TỔNG QUAN
Hệ thống quản lý tin nhắn Facebook tập trung. Khi khách hàng nhắn tin Messenger hoặc bình luận vào Facebook Page, tin nhắn hiện real-time lên dashboard web để admin xử lý, reply và quản lý.

API Endpoints
Method	Endpoint	Chức năng
GET	/health	Health check
GET/POST	/facebook/webhook	Facebook webhook
GET	/api/conversations	Danh sách hội thoại
GET	/api/conversations/:id/messages	Tin nhắn của 1 khách
POST	/api/conversations/:id/send	Gửi tin nhắn Messenger
POST	/api/conversations/:id/reply-comment	Reply bình luận
POST	/api/conversations/:id/send-from-comment	Gửi Messenger từ comment
POST	/api/conversations/:id/send-media	Gửi file/media
GET/POST/PUT/DELETE	/api/labels	Quản lý nhãn
GET/POST/DELETE	/api/customers/:id/labels	Nhãn của khách
GET/POST/PUT/DELETE	/api/quickreplies	Quản lý quick reply
POST	/api/translate	Dịch text
GET	/api/health	API health + socket clients
Socket.IO Events
Event	Hướng	Mô tả
new_message	Server → Client	Tin nhắn/comment mới
message_sent	Server → Client	Admin vừa gửi tin
connected	Server → Client	Xác nhận kết nối
ping/pong	Client ↔ Server	Keep-alive
DATABASE (PostgreSQL)
Bảng chính
Copycustomers          -- Khách hàng Facebook
messages           -- Lịch sử tin nhắn (có cột message_type: 'message'/'comment')
labels             -- Nhãn phân loại
customer_labels    -- Khách ↔ Nhãn (many-to-many)
quick_replies      -- Câu trả lời nhanh (có text_vi và text_en)
conversation_threads     -- Thread Telegram (legacy, không dùng)
conversation_mappings    -- Mapping Telegram (legacy, không dùng)
pending_messages         -- Tin chờ xác nhận (legacy, không dùng)
Cột quan trọng trong messages
Copysender_type    VARCHAR  -- 'customer' hoặc 'admin'
message_type   VARCHAR  -- 'message' hoặc 'comment'
translated_text TEXT    -- Bản dịch tiếng Việt (nếu có)
media_url      TEXT     -- URL media HOẶC comment_id (khi message_type='comment')
